## Tech Stack

- **Framework:** React 18 + React Router 7
- **Build:** Vite 7
- **Styling:** Tailwind CSS 4 + custom design system
- **Animation:** Framer Motion
- **Charts:** Recharts
- **Icons:** Phosphor Icons
- **Fonts:** Plus Jakarta Sans, Fira Code
- **Data:** Google Apps Script API (production metrics)
